GFI LanGuard

GFI LanGuard act as a virtual security consultant offering

  • Patch Management for Windows, Mac OS, Linux - GFI LanGuard enables complete patch management of security and non-security patches to Microsoft operating systems, Mac OS X, major Linux distributions and third-party applications. It can also automate patching for all major web browsers too.
  • Vulnerability Scanning for computers and mobile devices - Security audits check for over 60,000 vulnerability assessments using an extensive, industrial strength vulnerabilities database incorporating OVAL (11,500+ checks) and SANS Top 20 standards.

Key Benefits

  • Centralized Patch Management, vulnerability assessment and network auditing.
  • Automated patching for Windows , Mac OS, Linux operating systems and third party applications.
  • Over 60,000 vulnerability assessments carried out across network, including computers, smartphones, printer, routers, switches and virtual environment.
  • Assists with PCI DSS compliance and other security regulations (eg: HIPAA, CIPA, SOX, GLB/GLBA, PSN CoCo).
